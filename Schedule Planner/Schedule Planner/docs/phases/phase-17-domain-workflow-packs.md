# Phase 17 - Domain Workflow Packs

## 1. Muc tieu

- Tang chat luong tra loi theo tung nhom user cu the.
- Tranh mot bo workflow "one-size-fits-all" qua chung chung.
- Tang do dung de xuat vi workflow gan sat nganh canh.

## 2. Domain pack de xuat

1. Student pack
2. Office worker pack
3. Freelancer pack
4. Team lead pack
5. Personal productivity pack

## 3. Thanh phan moi pack

- Intent whitelist + workflow uu tien.
- Entity aliases theo domain language.
- Template clarification + template summary rieng.
- Rule priority/risk scoring theo domain.

## 4. Pham vi

Trong phase nay lam:

- Mechanism chon pack theo user profile/memory.
- A/B test pack strategy theo segment.
- Pack-specific evaluation dataset.

Chua lam:

- Auto-generate pack hoan toan bang self-learning.

## 5. Checklist

1. Tao config registry cho domain pack.
2. Route pack-aware trong `routeUserText`.
3. Add workflow prompt snippets theo pack.
4. Theo doi metric theo tung segment.

## 6. KPI va tieu chi hoan thanh

- User satisfaction tang theo tung segment.
- Clarification rate giam o cac domain da bat pack.
- Co quy trinh them domain pack moi trong < 1 tuan.
